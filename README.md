# 音乐播放器

一个基于 Web Audio API 的轻量级音乐播放器，支持实时频谱可视化与个性化设置。

## ✨ 特性

- **音频播放**：支持 MP3、WAV 等常见格式（通过本地文件加载）
- **同步歌词**：支持歌词同步滚动显示（仅当上传的文件中包含与音乐文件同名的歌词文件时支持显示歌词）
- **动态频谱**：基于 FFT 实时渲染音频频率柱状图
- **多种着色模式(待完善)**：
  - 单色（自定义颜色）
  - 能量着色（柱子越高越亮）
  - 彩虹渐变（低频→高频色彩过渡）
  - 热力图（红→黄→白）
- **交互式设置面板**：可调节频谱高度、颜色、着色模式
- **视觉细节**：顶部高光、平滑动画、深空主题 UI
- **纯前端实现**：无需后端，开箱即用

## 🛠 技术栈

- HTML5 + CSS3 + 原生 JavaScript (ES6+)
- Web Audio API（`AnalyserNode`, `AudioBufferSourceNode`）
- Canvas 2D API（频谱绘制）
- LocalStorage（保存用户设置）

## ▶️ 运行方式

1. 克隆本项目：
   ```bash
   git clone https://github.com/your-username/music-player.git
   cd music-player
   ```
2. 在浏览器中打开 index.html（推荐使用 Live Server 避免 CORS 问题）

    >💡 注意：由于浏览器安全策略，直接双击 index.html 可能无法加载本地音频文件。建议通过本地服务器运行。

## 🎛 使用说明

- 点击左下角 ⚙️ 设置图标

- 点击 “选择本地文件夹” 按钮上传本地音频、歌词（.lrc）

- 调整频谱高度、颜色及着色模式，设置自动保存

- 使用播放/暂停按钮控制播放
